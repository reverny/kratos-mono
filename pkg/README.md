# Shared Packages

Directory สำหรับเก็บ shared packages ที่ใช้ร่วมกันระหว่าง services ต่างๆ

## ตัวอย่าง

- `pkg/errors/` - Custom error types
- `pkg/middleware/` - Shared middleware
- `pkg/utils/` - Utility functions
- `pkg/logger/` - Logging utilities
- `pkg/auth/` - Authentication/Authorization helpers
